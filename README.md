# Вариационный вывод для анализа финансовых данных

**Предмет:** Методы искусственного интеллекта  
**Тема:** 3.2 — Вариационный вывод (Трейдинг)  
**Данные:** Московская биржа (MOEX ISS API)

---

## Постановка задачи

Дан временной ряд цен акций. Логарифмические доходности вычисляются как:

$$r_t = \ln\frac{S_t}{S_{t-1}}$$

**Цель:** оценить параметры распределения $r_t$ методом вариационного вывода и восстановить латентную волатильность.

---

## Математическая база

### Байесовский вывод

По теореме Байеса апостериорное распределение параметров:

$$p(\theta \mid r_{1:T}) = \frac{p(r_{1:T} \mid \theta)\; p(\theta)}{p(r_{1:T})}$$

Знаменатель $p(r_{1:T})$ аналитически **невычислим** для сложных моделей.

### Вариационный вывод (VI)

Идея: аппроксимировать $p(\theta \mid r_{1:T})$ простым распределением $q(\theta)$, минимизируя KL-дивергенцию:

$$q^*(\theta) = \arg\min_{q \in \mathcal{Q}}\; \text{KL}\bigl[q(\theta) \;\|\; p(\theta \mid r_{1:T})\bigr]$$

Эквивалентно максимизации ELBO (Evidence Lower BOund):

$$\text{ELBO}(q) = \mathbb{E}_{q}\bigl[\log p(r_{1:T},\, \theta)\bigr] - \mathbb{E}_{q}\bigl[\log q(\theta)\bigr]$$

### ADVI

Используется семейство нормальных распределений. Оптимизация — стохастическим градиентным спуском через автоматическое дифференцирование.

---

## Модель 1: Нормальное распределение

$$r_t \sim \mathcal{N}(\mu,\; \sigma^2)$$

**Априорные распределения:**

$$\mu \sim \mathcal{N}(0,\; 0.1^2) \qquad \sigma \sim \text{HalfNormal}(0.1)$$

**Функция правдоподобия:**

$$p(r_{1:T} \mid \mu, \sigma) = \prod_{t=1}^{T} \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(r_t - \mu)^2}{2\sigma^2}\right)$$

**Результат:** апостериорные распределения $q(\mu)$ и $q(\sigma)$ — колокольчики, отражающие неопределённость оценок.

---

## Модель 2: Стохастическая волатильность

**Мотивация:** волатильность реальных рынков непостоянна — в кризис высокая, в спокойное время низкая (кластеризация волатильности).

Скрытый процесс — лог-волатильность как случайное блуждание:

$$h_t = h_{t-1} + \sigma_h\,\varepsilon_t, \qquad \varepsilon_t \sim \mathcal{N}(0, 1)$$

Наблюдаемые доходности:

$$r_t \sim \mathcal{N}\!\left(0,\; e^{h_t}\right)$$

Мгновенная волатильность:

$$\sigma_t = e^{h_t / 2}$$

**Априорные распределения:**

$$\sigma_h \sim \text{HalfNormal}(0.5) \qquad h_0 \sim \mathcal{N}(-5,\; 2)$$

---

## Результаты (SBER, 2023–2025)

| Показатель | Значение |
|---|---|
| Торговых дней | 500 |
| Средняя доходность $\mu$ | ~+0.04% в день |
| Годовая волатильность $\sigma$ | ~24.1% |

**Модель 1:** 90% доверительный интервал для $\mu$ включает 0 — нет уверенного положительного тренда.

**Модель 2:** восстановленная траектория $\sigma_t$ фиксирует периоды повышенного риска. Пики волатильности соответствуют рыночным событиям.

**Вывод:** ADVI позволяет эффективно оценивать байесовские модели без аналитического вычисления апостериорного распределения. Модель стохастической волатильности превосходит нормальную, так как учитывает изменчивость риска во времени.

```bash

jupyter notebook main.ipynb
```

Или изменить тикер в `config.txt` и запустить `run.bat`.
