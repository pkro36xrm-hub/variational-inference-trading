# Вариационный вывод для анализа финансовых данных

**Предмет:** Методы искусственного интеллекта  
**Тема:** 3.2 — Вариационный вывод (Трейдинг)  
**Данные:** Московская биржа (MOEX ISS API)

---

## Постановка задачи

Дан временной ряд цен акций $\{S_t\}_{t=1}^{T}$.  
Логарифмические доходности: $r_t = \ln(S_t / S_{t-1})$

**Цель:** оценить параметры распределения $r_t$ методом вариационного вывода и восстановить латентную волатильность.

---

## Математическая база

### Байесовский вывод

По теореме Байеса апостериорное распределение параметров:

$$p(\theta | r_{1:T}) = \frac{p(r_{1:T} | \theta)\, p(\theta)}{p(r_{1:T})}$$

Знаменатель $p(r_{1:T}) = \int p(r_{1:T}|\theta)\,p(\theta)\,d\theta$ аналитически **невычислим** для сложных моделей.

### Вариационный вывод (VI)

Идея: аппроксимировать $p(\theta|r_{1:T})$ простым распределением $q(\theta) \in \mathcal{Q}$, минимизируя KL-дивергенцию:

$$q^*(\theta) = \arg\min_{q \in \mathcal{Q}}\; \text{KL}[q(\theta) \| p(\theta | r_{1:T})]$$

Эквивалентно максимизации ELBO (Evidence Lower BOund):

$$\text{ELBO}(q) = \mathbb{E}_{q}[\log p(r_{1:T}, \theta)] - \mathbb{E}_{q}[\log q(\theta)]$$

### ADVI (Automatic Differentiation Variational Inference)

Используется семейство нормальных распределений в пространстве без ограничений. Оптимизация ведётся стохастическим градиентным спуском через автоматическое дифференцирование.

---

## Модель 1: Нормальное распределение доходностей

$$r_t \sim \mathcal{N}(\mu,\, \sigma^2)$$

**Априорные распределения:**

$$\mu \sim \mathcal{N}(0,\; 0.1^2), \qquad \sigma \sim \text{HalfNormal}(0.1)$$

**Функция правдоподобия:**

$$p(r_{1:T}|\mu,\sigma) = \prod_{t=1}^{T} \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(r_t-\mu)^2}{2\sigma^2}\right)$$

**Результат:** апостериорные распределения $q(\mu)$ и $q(\sigma)$ — колокольчики, отражающие неопределённость оценок.

---

## Модель 2: Стохастическая волатильность

**Мотивация:** реальные рынки демонстрируют кластеризацию волатильности — в кризис она высокая, в спокойное время — низкая.

**Скрытый процесс** (лог-волатильность как случайное блуждание):

$$h_t = h_{t-1} + \sigma_h\,\varepsilon_t, \qquad \varepsilon_t \sim \mathcal{N}(0,1)$$

**Наблюдаемые доходности:**

$$r_t \sim \mathcal{N}\!\left(0,\; e^{h_t}\right)$$

**Мгновенная волатильность:**

$$\sigma_t = e^{h_t/2}$$

**Априорные распределения:**

$$\sigma_h \sim \text{HalfNormal}(0.5), \qquad h_0 \sim \mathcal{N}(-5,\;2)$$

---

## Результаты (Сбербанк, SBER, 2023–2025)

| Показатель | Значение |
|---|---|
| Период данных | 2023-06-14 — 2025-05-07 |
| Торговых дней | 500 |
| Средняя доходность μ | ~+0.04% в день |
| Годовая волатильность σ | ~24.1% |

**Вывод по Модели 1:** 90% доверительный интервал для μ включает 0 — нет уверенного положительного тренда.

**Вывод по Модели 2:** восстановленная траектория $\{\sigma_t\}$ фиксирует периоды повышенного риска, соответствующие рыночным событиям. Пики волатильности совпадают с периодами рыночной нестабильности.

**Общий вывод:** Вариационный вывод (ADVI) позволяет эффективно оценивать параметры байесовских моделей без аналитического вычисления апостериорного распределения. Модель стохастической волатильности превосходит нормальную модель, так как учитывает изменчивость риска во времени — ключевое свойство реальных финансовых данных.

## Структура репозитория

```
├── main.ipynb          # Нотбук с кодом, графиками и пояснениями
├── main.py             # То же самое в виде скрипта
├── requirements.txt    # Зависимости
├── config.txt          # Тикер для анализа (меняй здесь)
└── run.bat             # Запуск одним двойным кликом (Windows)
```

## Как запустить

```bash
pip install -r requirements.txt
jupyter notebook main.ipynb
```

Или изменить тикер в `config.txt` и запустить `run.bat`.
