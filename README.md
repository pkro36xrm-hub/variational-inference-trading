# Вариационный вывод: анализ финансовых данных

## Описание проекта

Проект посвящён применению **вариационного вывода** (Variational Inference) к финансовым данным. 
На основе исторических данных котировок акций строятся байесовские модели, 
позволяющие оценить распределение параметров и сделать вероятностные предсказания.

## Структура проекта

```
variational_inference/
├── main.py              # Главный скрипт — запускает всё
├── requirements.txt     # Зависимости
├── README.md            # Описание проекта
└── results/             # Папка с графиками (создаётся автоматически)
```

## Используемые технологии

- **Python 3.10+**
- **PyMC 5** — библиотека вероятностного программирования
- **ArviZ** — визуализация результатов байесовского вывода
- **yfinance** — загрузка финансовых данных
- **matplotlib** — построение графиков

## Установка и запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить анализ
python main.py
```

## Методология

### 1. Данные
Используются дневные котировки акций (цены закрытия) за последние 3 года.
Из цен рассчитываются **логарифмические доходности** — стандартная метрика в финансах.

### 2. Модели

#### Модель 1: Нормальное распределение доходностей
Простая модель, предполагающая что доходности распределены нормально:
- `μ` (mu) — средняя доходность
- `σ` (sigma) — волатильность (разброс)

#### Модель 2: Стохастическая волатильность
Более сложная модель, учитывающая что волатильность **меняется со временем**:
- В спокойные периоды — волатильность низкая
- В кризисы — волатильность высокая

### 3. Вариационный вывод
Вместо точного вычисления апостериорного распределения (что невозможно аналитически), 
используется **ADVI** (Automatic Differentiation Variational Inference) — 
метод аппроксимации, который подбирает параметрическое распределение, 
наиболее близкое к истинному апостериорному.

### 4. Результаты
- Апостериорные распределения параметров
- Предсказания с доверительными интервалами
- Сравнение моделей
